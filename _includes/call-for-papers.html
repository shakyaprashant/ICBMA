The 2nd International Conference on Big Data, Machine Learning & their Applications (ICBMA-2021) is the premier forum for the presentation of new advances and research results in the fields of Big Data, Machine learning & Their Applications. The conference will bring together leading researchers, engineers and scientists in the domain of interest from around the world. All submitted original papers will be peer reviewed before final acceptance.

Topics of interest for submission include:

Track I: Big Data

  1. Algorithms and Programming Techniques for Big Data Processing
  2. Architecture and Security Issues in Big data
  3. Cloud computing implementation of big data
  4. Optimization and Analytics issues in big data
Track-II: Machine Learning

  1. Machine learning applications and Methods in clustering & classification
  2. Deep learning Algorithms and its applications
  3. Support Vector Machine & its application
  4. Convolution neural Network & its application
  5. Bayesian Model, HMM and its applications
  6. Variants of All existing Models
Track III: Bio Inspired Algorithms

  1. Genetic Algorithms & its application
  2. Multi objective optimization & its application
  3. Computational intelligence & its application
  4. Evolutionary Algorithms & its application
  5. Combinatorial Optimization & its application
  6. Swarm Intelligence Algorithm & its application
Track IV: Artificial Intelligence

  1. Robotics & Automation
  2. Natural Language Processing
  3. Neural Network and Fuzzy logic
  4. Decision Support Systems
  5. Intelligent Image Processing Algorithms
  6. Computer Vision
  7. Pattern Recognition
Track V: Applications of Big Data, Machine Learning & Bio Inspired Algorithms in

  1. Precision Agriculture
  2. Smart city
  3. Fog Computing
  4. Cloud Computing
  5. Systems biology
  6. IoT
  7. Medical and Health informatics
  8. Any other applications of Big Data, Machine Learning and AI.
